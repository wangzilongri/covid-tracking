# TLGRF Manuscript Analysis and Benchmarks

This folder contains the code for benchmarking `TLGRF` against various
other methods as brought up in the manuscript and the rebuttal letter

The key folders to look out for are the `benchmark_*` folders.

`generate_benchmark_TLGRF_dataset` contains the code needed to generate
a clean version of the historical estimations by `TLGRF`.

For readability purposes, the results are presented as Jupyter notebooks
that have were ran from scratch
